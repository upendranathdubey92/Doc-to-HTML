📋 BASIC RULES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✓ Section names MUST be ALL CAPS with underscores
✓ Every section NEEDS [START] and [END] markers
✓ Keep markers on separate lines
✓ One blank line between sections (optional but recommended)

❌ Wrong: [mcb_hero_banner_with_form]
✓ Right: [MCB_HERO_BANNER_WITH_FORM]

❌ Wrong: [MCB_SERVICES_GRID] content here [MCB_SERVICES_GRID END]
✓ Right:
[MCB_SERVICES_GRID]
content here
[MCB_SERVICES_GRID END]

🎨 FORMATTING TIPS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
✓ You CAN use:
  • Bold text
  • Italic text
  • Hyperlinks
  • Bullet points (●)
  • Numbers
  • Multiple paragraphs

❌ AVOID:
  • Word comments (filtered automatically but causes clutter)
  • Track changes (will appear in output)
  • Tables inside sections (may not parse correctly)
  • Images (handled separately via HTML)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📖 TYPICAL PAGE STRUCTURE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

For a standard service page, use this order:

1. MCB_HERO_BANNER_WITH_FORM      ← Top of page with form
2. MCB_TRUSTED_BRANDS             ← Logo bar
3. MCB_SERVICES_GRID              ← What you offer
4. MCB_STATS_BY_NUMBERS           ← Company stats
5. MCB_CTA_SECTION                ← First conversion point
6. MCB_BENEFITS_GRID              ← Why choose us
7. MCB_PORTFOLIO_PROJECTS         ← Case studies
8. MCB_PROCESS_STEPS              ← How we work
9. MCB_TECHNOLOGY_STACK           ← Tech we use
10. MCB_HIRE_DEVELOPERS           ← Team augmentation
11. MCB_ENGAGEMENT_MODELS         ← Pricing options
12. MCB_TESTIMONIAL_CLUTCH_SLIDER ← Social proof
13. MCB_FAQ_ACCORDION             ← Common questions
14. MCB_CTA_INLINE                ← Final conversion

Not all pages need all sections. Choose what makes sense!

2. Save as .docx

5. Copy output HTML
6. Include MCB-style.css in your page
7. Done!

🆘 TROUBLESHOOTING
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Problem: Section not detected
✓ Check section name is ALL CAPS
✓ Verify both START and END markers exist
✓ Make sure markers are on their own lines

Problem: Content looks weird
✓ Remove Word comments before uploading
✓ Check for hidden formatting
✓ Try "Clear Formatting" in Word first

Problem: Missing content in output
✓ Verify content is between [START] and [END] markers
✓ Check if section name is spelled correctly
✓ Look at browser console for errors
